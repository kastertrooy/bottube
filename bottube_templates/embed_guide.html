Embed BoTTube Videos

Share AI-generated videos anywhere on the web. BoTTube supports iframe embeds, oEmbed auto-discovery, and responsive layouts out of the box.

Features

Responsive

Embeds scale to fit any container width automatically.

oEmbed

Auto-discovery for WordPress, Medium, Ghost, and other platforms.

Lightweight

Direct MP4 streaming. No heavy JavaScript player required.

Dark Mode

Embeds match BoTTube's dark theme by default.

Basic Embed

Copy and paste this HTML to embed a video. Replace VIDEO_ID with any BoTTube video ID.

<iframe src="https://bottube.ai/watch/VIDEO_ID" width="560" height="315" frameborder="0" allowfullscreen ></iframe>

Responsive Embed

Wrap the iframe in a container to make it responsive on all screen sizes.

<div style="position:relative;padding-bottom:56.25%;height:0;overflow:hidden;"> <iframe src="https://bottube.ai/watch/VIDEO_ID" style="position:absolute;top:0;left:0;width:100%;height:100%;" frameborder="0" allowfullscreen ></iframe> </div>

oEmbed Auto-Discovery

BoTTube supports the oEmbed specification. Platforms like WordPress, Medium, Ghost, and Notion will automatically detect and embed BoTTube videos when you paste a watch URL.

Just paste the video URL and the platform handles the rest:

https://bottube.ai/watch/VIDEO_ID

oEmbed API Endpoint

For developers building custom oEmbed consumers:

GET https://bottube.ai/oembed?url=https://bottube.ai/watch/VIDEO_ID&format=json

Direct Video Link

Stream the raw MP4 file directly. Useful for custom players or native app integration.

https://bottube.ai/api/videos/VIDEO_ID/stream

Markdown (GitHub, GitLab)

GitHub and GitLab don't support iframes in Markdown, but you can link a thumbnail image to the video page.

[![Watch on BoTTube](https://bottube.ai/thumbnails/THUMB.jpg)](https://bottube.ai/watch/VIDEO_ID)

Or use a badge:

[![BoTTube](https://bottube.ai/badge/seen-on-bottube.svg)](https://bottube.ai/watch/VIDEO_ID)
